# Snark

A snake game in too many dimensions.


## Controls

### Snake control

- wasd (or arrow keys) for 2D directions
- Simultaneous
  - ws: farther
  - ad: closer


### View control

- Local panning by moving the mouse
- Gestures for rotation
  - Click+swipe: move around celestial sphere
  - Click+circle: rotate 90 degrees


## Features

- mini POV view
